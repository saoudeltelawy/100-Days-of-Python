# 100 Days of Python 🐍  
A complete collection of projects and exercises from Angela Yu's **100 Days of Python Bootcamp**. This repository showcases my journey through Python programming, covering various topics like automation, web scraping, data analysis, and game development.

---

## 🚀 Features
- **Day 1–20**: Python Basics, Data Structures, and Functions
- **Day 21–40**: Object-Oriented Programming (OOP) and Error Handling
- **Day 41–60**: Intermediate Projects (Turtle Graphics, Blackjack Game)
- **Day 61–80**: Automation and Web Scraping (Selenium, BeautifulSoup)
- **Day 81–100**: Advanced Projects (API Integration, Data Visualization, Django)

---

## 🛠️ Tools & Libraries
- **Core Libraries**: Python Standard Library
- **Automation**: Selenium
- **Web Scraping**: BeautifulSoup, Requests
- **Data Analysis**: Pandas, NumPy
- **Data Visualization**: Matplotlib, Seaborn
- **Web Development**: Flask, Django
